Spørsmål og svar
Vanlige spørsmål
Trenger vi mikrofon eller spesielt utstyr?▼
Nei! Alle moderne telefoner, nettbrett og laptoper har innebygd mikrofon. AirPods og trådløse øreplugger fungerer også optimalt.
Hvilke nettlesere fungerer best?▼
Talegjenkjenning fungerer best i Chrome og Edge. Safari på iPhone og iPad fungerer også. Firefox støtter ikke talegjenkjenning.
Hvor nøyaktig er talegjenkjenningen?▼
Talegjenkjenning fungerer bra for vanlige norske ord, men kan slite med sjeldne ord eller dialekter. Barnet kan alltid prøve setningen om igjen så mange ganger det trengs. For best nøyaktighet anbefales Chrome på telefon eller nettbrett.
Fungerer Leselab for barn med dysleksi?▼
Ja! Setningsbasert lesing er særlig nyttig for barn med dysleksi siden det bryter teksten ned i håndterbare biter.
Kan jeg avbestille når som helst?▼
Ja, du kan avbestille når som helst uten bindingstid.
